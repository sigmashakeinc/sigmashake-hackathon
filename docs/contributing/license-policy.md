# License Policy

No open-source license has been selected in this scaffold.

Until maintainers choose and add a repository `LICENSE`, external contributors should only submit changes they are authorized to contribute and understand that maintainers may not merge public contributions until licensing is resolved.

Recommended maintainer choices before broad public contribution:

- Add the intended repository license.
- State whether inbound contributions are under the same license.
- Add any contributor agreement requirements if Sigma Shake needs them.
